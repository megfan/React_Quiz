# ReactQuiz
A multiple choice quiz with React about react made as a final project at CodersLab bootcamp.

View Demo:
[React Quiz](https://megfan.github.io)

Getting Started

    $ git clone https://github.com/davidrayoussef/react-quiz.git
    
    $ cd react-quiz
    
    $ npm install
    
    $ npm start



App Structure:

            ├── App

            |   |    ├── Answers
            |   |    └── Stopwatch
